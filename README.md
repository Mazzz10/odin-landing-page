## 🛠️ Features & Implementation
- **Pure HTML5 & CSS3:** Structured semantically without external CSS frameworks or layout systems.
- **Flexbox Layout Architecture:** Built cleanly using alignment rules across 4 main sections (Hero, Features/Information grid, Testimonial/Quote block, and Call to Action panel).
- **Pixel-Approximated Design:** Styled meticulously following custom design color sheets and typography specs.
